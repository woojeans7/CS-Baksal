# 네트워크의 분류

네트워크를 분류할 수 있는 방법이 2가지 있다.

1. 범위에 따른 분류
2. 메시지 교환 방식에 따른 분류

## 범위에 따른 분류

크게 3가지로 나뉜다.

- LAN
- WAN
- MAN

### LAN

LAN(Local Area Network)은 가까운 지역을 연결한 근거리 네트워크로, 홈용, 사무용 같이 비교적 소규모의 네트워크를 말한다.

- 이더넷(Ethernet), WiFi
- 높은 속도, 낮은 에러율, 낮은 비용

<br>

### WAN

WAN(Wide Area Network)은 먼 거리에 있는 네트워크를 연결하는 광역 네트워크이다.

- 인터넷이 WAN으로 분류되며, 다른 LAN에 속한 호스트와 메시지를 주고 받아야 할 때 필요하다.
- 특별한 경우가 아니면 직접 구축할 수 없음. → ISP(Internet Service Provider)를 통해 회선을 임대해서 사용. (국내 대표 ISP : KT, LG유플러스, SK 브로드밴드)

<br>

### MAN

MAN(Metro Area Network)은 한 도시 정도를 연결하고 관리하는 네트워크이다.

- 일반적으로 도시 자체 인프라를 구성하면 MAN, 도시에서 통신사 인프라 기반으로 구축하면 WAN으로 구분한다.
- LAN, WAN이 더 중요해서 잘 안쓰이는 편이다.

<br>

### 범위별 네트워크 비교

| **구분**  | **LAN**                | **MAN**        | **WAN**     |
| --------- | ---------------------- | -------------- | ----------- |
| 범위      | 건물/캠퍼스            | 도시           | 국가/대륙   |
| 속도      | 높음 (Gbps)            | 중간           | 낮음 (Mbps) |
| 지연시간  | 매우 낮음              | 낮음           | 높음        |
| 소유/관리 | 개인/기업              | 통신사/지자체  | 통신사(ISP) |
| 예시      | 회사 내부망, 가정 WiFi | 도시 공공 WiFi | 인터넷      |

<br>

## 메시지 교환 방식에 따른 분류

### 회선 교환 방식 (회선 교환 네트워크)

회선 교환 방식(Circuit Switching)은 메시지를 주고 받기 전 **회선(circuit)을 미리 설정**한 뒤, 해당 회선을 통해 메시지를 주고 받는 방식이다.

![[Pasted image 20251102204705.png]]

###### 이미지 출처 : [code360](https://www.naukri.com/code360/library/switching-in-computer-networks)

- 회선 스위치 : 호스트 사이에 일대일 전송로를 확보하는 네트워크 장비
- 장점 : 안정적인 전송, **일정한 전송 속도** (전송되는 정보의 양이 비교적 일정)
- 단점 : **회선 낭비** (회선의 이용 효율이 낮아짐)

<br>

### 패킷 교환 방식 (패킷 교환 네트워크)

패킷 교환 방식(Packet Switching)는 메시지를 패킷(Packet)이라는 단위로 쪼개어 전송하는 방식이다.
쪼개어 전송된 패킷들은 수신지에서 재조립됨.

![[Pasted image 20251102204815.png]]

###### 이미지 출처 : [code360](https://www.naukri.com/code360/library/switching-in-computer-networks)

- **패킷** : 패킷 교환 네트워크상에서 송수신되는 메시지의 단위
- 패킷 스위치 : 패킷의 송수신지를 시별, 패킷이 이동할 최적의 경로를 결정하는 장비 (라우터, 스위치)
- **전송로의 이용 효율이 높아 회선 교환 방식의 문제점을 해결**
- 현대 인터넷은 대부분 패킷 교환 방식을 이용

<br>

### 회선 교환 vs 패킷 교환 비교

| **구분**  | **회선 교환**  | **패킷 교환**  |
| --------- | -------------- | -------------- |
| 연결 방식 | 전용 회선 할당 | 회선 공유      |
| 자원 활용 | 낮음 (독점)    | 높음 (공유)    |
| 전송 속도 | 일정           | 가변적         |
| 지연 시간 | 일정           | 가변적         |
| 예시      | 전화망         | 인터넷(TCP/IP) |
